f(2x + 1) = (2x + 1)² - 3(2x + 1) - 1
f(x) = x² - 3x - 1

g(x - 1) = 2x + 5
g(x - 1) = 2(x - 1) + 7
g(x) = 2x + 7

(f+g) (x + 1) = f(x + 1) + g(x + 1)
= (x + 1)² - 3(x + 1) - 1 + 2(x + 1) + 7
= x² + 2x + 1 - 3x - 3 - 1 + 2x + 2 + 7
= x² + 5x + 6

18. Haasil perkalian dari bentuk aljabar berikut adalah1. (x + 2) (x + 3)2. (2x + 3) (x2 + 2x – 5)​


Jawab:

1. (x + 2) (x + 3)

= x ( x + 3 ) + 2 ( x + 3 )

= x² + 3x + 2x + 6

= x² + 5x + 6

2. Kita selesaikan dengan menggunakan sifat distributif

(2x + 3) (x2 + 2x – 5)

= 2x (x² + 2x – 5 ) + 3 (x² + 2x – 5)

= 2x³ + 4x² – 10x + 3x² + 6x – 15

= 2x³ + 4x² + 3x² – 10x + 6x – 15

= 2x³ + 7x² – 4x – 15

(x + a)(x + b) ➡ x(x + b) + a(x + b)

1. (x + 2) (x + 3)

= x(x + 3) + 2(x + 3)

= x² + 3x + 2x + 6

= x² + 5x + 6

2. (2x + 3) (x² + 2x – 5)

= 2x³ + 4x² – 10x + 3x² + 6x – 15

= 2x³ + (4x² + 3x²) – 10x + 6x - 15

= 2x³ + 7x² – 4x – 15

20. haasil invers dari (g^-1of)^-1(x) jika f^-1(x) = 3-x/4x-5 dan g(x) =4 adalahtolong bantuin ya :") ​


Jawaban:

f-¹(x) = 3 - x / 4x - 5

g(x) = 4x

cari f(x)

misal f-¹(x) = y

y = 3 - x / 4x - 5

y ( 4x - 5 ) = 3 - x

4xy - 5y = 3 - x

4xy + x = 5y + 3

x ( 4y + 1 ) = 5y + 3

x = 5y + 3 / 4y + 1

f(x) = 5x + 3 / 4x + 1

(g-¹ o f-¹)(x) = (fog)-¹(x)

cari (fog)(x)

= 5(4x) + 3 / 4(4x) + 1

= 20x + 3 / 16x + 1

misal (fog)(x) = y

y = 20x + 3 / 16x + 1

y ( 16x + 1 ) = 20x + 3

16xy + y = 20x + 3

16xy - 20x = 3 - y

x ( 16y - 20 ) = 3 - y

x = 3 - y / 16y - 20

(fog)-¹(x) = 3 - x / 16x - 20

Penjelasan dengan langkah-langkah:

(g^-1of)^-1(x) = (f^-1og)

= 3-(4x)/4(4x)-5
